#  Landing Page — Cyber Segurança | InterWeg

Landing page institucional para captação de leads interessados nos produtos da InterWeg Seguros.

O objetivo é educar o mercado, qualificar empresas interessadas e direcionar os contatos para atendimento automatizado (bot / WhatsApp / CRM).

---

##  Objetivo

- Gerar **leads qualificados** (empresas) para Seguro Cyber
- Direcionar usuários para:
  - Bot de atendimento (ManyChat / WhatsApp / Instagram DM)
  - Ou formulário de contato / agendamento
- Posicionar a InterWeg como **especialista em proteção cibernética**
- Permitir **segmentação de campanhas** e **classificação de leads**

---

##  Público-Alvo

- Empresas (PJ), principalmente:
  - Pequenas e médias empresas
  - E-commerces
  - Escritórios, clínicas, contabilidade, imobiliárias, etc.
- Decisores:
  - Donos
  - Gestores administrativos/financeiros
  - Responsáveis por TI/Operações

---

##  Jornada do Usuário

1. Usuário chega via campanha (Google, Instagram, LinkedIn, etc.)
2. Visualiza:
   - O problema (riscos digitais)
   - A solução (seguro + proteção)
   - A autoridade (InterWeg)
3. Entende:
   - O que é Cyber Segurança
   - Por que a empresa precisa
   - Quais riscos corre sem proteção
4. Clica no CTA:
   - “Quero proteger minha empresa”
5. É direcionado para:
   - Bot / WhatsApp / formulário
6. Lead é classificado:
   - Tipo de empresa
   - Tipo de risco
   - Produto de interesse

---

##  Estrutura da Página

- **Hero**
  - Headline principal
  - Subheadline explicativa
  - CTA principal
- **Seção: O Problema**
  - Riscos cibernéticos (ataques, vazamentos, paralisações)
- **Seção: A Solução**
  - O que o Seguro Cyber cobre
  - Benefícios para a empresa
- **Seção: Para Quem É**
  - Tipos de empresas atendidas
- **Seção: Como Funciona**
  - Diagnóstico → Cotação → Proteção
- **Prova / Autoridade**
  - InterWeg, parceiros, credibilidade
- **CTA Final**
  - Botão para contato / automação
